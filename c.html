O que é C?

C é uma linguagem de programação de propósito geral, imperativa, procedural e compilada. Criada por Dennis Ritchie em 1972, é uma das linguagens mais influentes da história da computação.

Características principais:

  • Simples e Eficiente: Sintaxe clara e direta com mínima abstração
  • Rápida: Código compilado que executa com alta performance
  • Portável: Funciona em praticamente qualquer plataforma
  • Baixo Nível: Acesso direto à memória através de ponteiros
  • Influente: Base para linguagens como C++, C#, Java e JavaScript

C é essencial para sistemas operacionais, bancos de dados, compiladores e aplicações de tempo real.

Configuração do Ambiente

Para programar em C, você precisa de um compilador e um editor de texto ou IDE.

Passos básicos:

  1. Instalar um compilador (GCC, Clang ou MinGW)
  2. Escolher um editor ou IDE
  3. Criar e compilar seu primeiro programa
  4. Executar o programa compilado

Verificando a instalação:

gcc --version
clang --version

Compiladores Populares

  • GCC: GNU Compiler Collection (Linux, Windows)
  • Clang: Compilador moderno (macOS, Linux)
  • MinGW: GCC para Windows
  • MSVC: Microsoft Visual C++

IDEs e Editores

  • Code::Blocks
  • Dev-C++
  • Visual Studio Code
  • CLion (JetBrains)

Primeiro Programa em C

Vamos criar o tradicional programa "Hello World" em C.

Estrutura básica:

#include <stdio.h>

int main() {
    printf("Olá, Mundo!\n");
    return 0;
}

Para compilar e executar:

gcc hello.c -o hello
./hello

Anatomia do programa

  • #include <stdio.h>: Inclui a biblioteca padrão de I/O
  • int main(): Função principal (ponto de entrada)
  • printf(): Função para imprimir texto
  • return 0;: Retorna sucesso ao sistema operacional

Convenções de código

  • Nomes de funções: snake_case (minha_funcao)
  • Nomes de constantes: UPPER_CASE (MINHA_CONSTANTE)
  • Nomes de variáveis: snake_case (minha_variavel)
  • Indentação: 2 ou 4 espaços

Sintaxe e Tipos de Dados

C possui tipos de dados primitivos bem definidos e uma sintaxe clara.

Variáveis e Tipos

int idade = 25;        // Inteiro
float altura = 1.75;   // Ponto flutuante
char inicial = 'J';    // Caractere
double pi = 3.14159;   // Dupla precisão
unsigned int id = 100; // Sem sinal

Operadores

int a = 10, b = 3;
int soma = a + b;      // 13
int subtracao = a - b; // 7
int multiplicacao = a * b; // 30
int divisao = a / b;   // 3
int modulo = a % b;    // 1

Ponteiros - Conceito Fundamental

Ponteiros são uma das características mais poderosas de C, permitindo acesso direto à memória.

int x = 10;
int *ptr = &x;  // Ponteiro para x
printf("%d\n", *ptr);  // Imprime 10
printf("%p\n", ptr);   // Imprime endereço de memória
