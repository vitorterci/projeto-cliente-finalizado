O que é C++?

C++ é uma linguagem de programação compilada, multiparadigma que estende C com recursos de programação orientada a objetos. Criada por Bjarne Stroustrup em 1985, é uma das linguagens mais poderosas e versáteis.

Características principais:

  • Multiparadigma: Suporta programação procedural, orientada a objetos e funcional
  • Alto Desempenho: Compilação eficiente e execução rápida
  • Controle Total: Acesso direto à memória e recursos do sistema
  • STL Poderosa: Biblioteca padrão com containers e algoritmos
  • Compatibilidade: Retro-compatível com C

C++ é usado em sistemas operacionais, motores de jogos, aplicações financeiras e software de alto desempenho.

Configuração do Ambiente

Para programar em C++, você precisa de um compilador compatível e um editor ou IDE.

Passos básicos:

  1. Instalar um compilador (G++, Clang++ ou MSVC)
  2. Escolher um editor ou IDE
  3. Criar seu primeiro programa C++
  4. Compilar e executar

Verificando a instalação:

g++ --version
clang++ --version

Compiladores Populares

  • G++: GNU C++ Compiler
  • Clang++: Compilador moderno e rápido
  • MSVC: Microsoft Visual C++
  • Intel C++: Compilador otimizado

IDEs Recomendadas

  • Visual Studio
  • CLion (JetBrains)
  • Code::Blocks
  • Visual Studio Code

Primeiro Programa em C++

Vamos criar o programa "Hello World" em C++.

Estrutura básica:

#include <iostream>

int main() {
    std::cout << "Olá, Mundo!" << std::endl;
    return 0;
}

Para compilar e executar:

g++ hello.cpp -o hello
./hello

Diferenças de C

  • #include <iostream>: Biblioteca de entrada/saída
  • std::cout: Saída padrão (mais moderno que printf)
  • <<: Operador de inserção
  • Suporte a namespaces

Classes em C++

C++ permite criar classes para estruturar dados e comportamentos.

class Pessoa {
private:
    std::string nome;
    int idade;

public:
    Pessoa(std::string n, int i) : nome(n), idade(i) {}
    
    void apresentar() {
        std::cout << "Olá, meu nome é " << nome << std::endl;
    }
};

STL - Standard Template Library

A STL fornece containers, iteradores e algoritmos poderosos.

#include <vector>
#include <algorithm>

std::vector<int> numeros = {3, 1, 4, 1, 5};
std::sort(numeros.begin(), numeros.end());

for (int num : numeros) {
    std::cout << num << " ";
}
