O que é C#?

C# é uma linguagem de programação moderna, orientada a objetos, desenvolvida pela Microsoft como parte da plataforma .NET. Combina a simplicidade de linguagens de alto nível com o poder de linguagens compiladas.

Características principais:

  • Orientada a Objetos: Suporte completo a POO com herança, polimorfismo e encapsulamento
  • Segura: Gerenciamento automático de memória com garbage collection
  • Moderna: Recursos como LINQ, async/await e pattern matching
  • Multiplataforma: .NET Core permite executar em Windows, Linux e macOS
  • Produtiva: Sintaxe clara e ferramentas poderosas

C# é usado em desenvolvimento web (ASP.NET), desktop (WPF), mobile (Xamarin) e jogos (Unity).

Configuração do Ambiente

Para programar em C#, você precisa instalar o .NET SDK e um editor ou IDE.

Passos básicos:

  1. Instalar o .NET SDK
  2. Instalar Visual Studio ou Visual Studio Code
  3. Criar um novo projeto
  4. Executar seu programa

Verificando a instalação:

dotnet --version

Plataformas .NET

  • .NET Framework: Windows apenas (legado)
  • .NET Core: Multiplataforma (moderno)
  • .NET 5+: Unificação do .NET Core
  • Mono: Implementação open-source

IDEs Recomendadas

  • Visual Studio (Completo)
  • Visual Studio Code
  • JetBrains Rider
  • Monodevelop

Primeiro Programa em C#

Vamos criar o programa "Hello World" em C#.

Estrutura básica:

using System;

class Program {
    static void Main() {
        Console.WriteLine("Olá, Mundo!");
    }
}

Para compilar e executar:

dotnet new console -n HelloWorld
cd HelloWorld
dotnet run

Estrutura do Projeto

  • using: Importa namespaces
  • class Program: Define uma classe
  • static void Main(): Ponto de entrada
  • Console.WriteLine(): Imprime texto

Convenções de Código

  • Nomes de classes: PascalCase (MinhaClasse)
  • Nomes de métodos: PascalCase (MeuMetodo)
  • Nomes de variáveis: camelCase (minhaVariavel)
  • Constantes: UPPER_CASE (MINHA_CONSTANTE)

Tipos de Dados e Variáveis

C# possui um sistema de tipos forte e seguro.

int idade = 25;           // Inteiro
double altura = 1.75;     // Ponto flutuante
string nome = "João";     // Texto
bool ativo = true;        // Booleano
decimal preco = 99.99m;   // Decimal (financeiro)

Programação Orientada a Objetos

C# oferece suporte completo a POO com classes, herança e polimorfismo.

class Pessoa {
    public string Nome { get; set; }
    public int Idade { get; set; }
    
    public void Apresentar() {
        Console.WriteLine($"Olá, meu nome é {Nome}");
    }
}

var pessoa = new Pessoa { Nome = "Maria", Idade = 30 };
pessoa.Apresentar();

LINQ - Language Integrated Query

LINQ permite consultas integradas na linguagem.

var numeros = new[] { 1, 2, 3, 4, 5 };
var pares = numeros.Where(n => n % 2 == 0);
var dobrados = pares.Select(n => n * 2);

foreach (var n in dobrados) {
    Console.WriteLine(n);
}
