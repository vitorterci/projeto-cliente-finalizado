O que é Go?

Go (Golang) é uma linguagem de programação compilada, criada pelo Google em 2009. É projetada para ser simples, eficiente e ideal para programação concorrente e sistemas distribuídos.

Características principais:

  • Simples: Sintaxe clara e minimalista
  • Rápida: Compila para código nativo muito rápido
  • Concorrência: Goroutines e channels para concorrência fácil
  • Multiplataforma: Compila para Windows, Linux, macOS
  • Biblioteca Padrão Poderosa: Inclui tudo que você precisa

Go é usado em Docker, Kubernetes, Prometheus e muitos serviços de cloud computing.

Configuração do Ambiente

Para programar em Go, você precisa instalar o Go SDK e um editor.

Passos básicos:

  1. Baixar e instalar Go de golang.org
  2. Configurar GOPATH
  3. Instalar um editor
  4. Criar seu primeiro programa

Verificando a instalação:

go version
go env

Estrutura de Projeto

  • GOPATH: Diretório raiz dos projetos Go
  • src: Código-fonte
  • bin: Binários compilados
  • pkg: Pacotes compilados

Editores Recomendados

  • Visual Studio Code
  • GoLand (JetBrains)
  • Vim/Neovim
  • Sublime Text

Primeiro Programa em Go

Vamos criar o programa "Hello World" em Go.

Estrutura básica:

package main

import "fmt"

func main() {
    fmt.Println("Olá, Mundo!")
}

Para compilar e executar:

go run hello.go
go build hello.go
./hello

Estrutura do Programa

  • package main: Define o pacote
  • import: Importa pacotes
  • func main(): Função principal
  • fmt.Println(): Imprime texto

Convenções de Código

  • Nomes exportados: PascalCase (MinhaFuncao)
  • Nomes privados: camelCase (minhaFuncao)
  • Nomes de variáveis: camelCase (minhaVariavel)
  • Constantes: UPPER_CASE (MINHA_CONSTANTE)

Funções em Go

Funções em Go são simples e poderosas.

func saudacao(nome string) string {
    return "Olá, " + nome
}

func soma(a, b int) int {
    return a + b
}

// Múltiplos retornos
func divisao(a, b int) (int, error) {
    if b == 0 {
        return 0, fmt.Errorf("divisão por zero")
    }
    return a / b, nil
}

Goroutines - Concorrência Simples

Goroutines são a forma elegante de Go para concorrência.

package main

import (
    "fmt"
    "time"
)

func tarefa(id int) {
    for i := 0; i < 3; i++ {
        fmt.Printf("Tarefa %d - Iteração %d\n", id, i)
        time.Sleep(time.Second)
    }
}

func main() {
    go tarefa(1)
    go tarefa(2)
    time.Sleep(4 * time.Second)
}

Channels - Comunicação entre Goroutines

Channels permitem que goroutines se comuniquem de forma segura.

package main

import "fmt"

func main() {
    ch := make(chan string)
    
    go func() {
        ch <- "Olá do goroutine!"
    }()
    
    mensagem := <-ch
    fmt.Println(mensagem)
}
