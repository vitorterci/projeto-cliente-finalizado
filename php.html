O que é PHP?

PHP (Hypertext Preprocessor) é uma linguagem de script de servidor de propósito geral, especialmente adequada para desenvolvimento web. Criada por Rasmus Lerdorf em 1995, é uma das linguagens mais usadas na internet.

Características principais:

  • Fácil de Aprender: Sintaxe simples e direta
  • Lado do Servidor: Executa no servidor, não no navegador
  • Dinâmico: Gera conteúdo HTML dinamicamente
  • Integrado com HTML: Pode ser embutido em HTML
  • Suporte a Banco de Dados: Integração nativa com MySQL, PostgreSQL, etc.

PHP é usado em WordPress, Drupal, Magento e milhões de sites em todo o mundo.

Configuração do Ambiente

Para programar em PHP, você precisa de um servidor web, PHP e um editor de texto.

Passos básicos:

  1. Instalar um servidor local (XAMPP, LAMP ou WAMP)
  2. Escolher um editor (VS Code, Sublime Text, PhpStorm)
  3. Criar arquivos .php
  4. Acessar via navegador

Verificando a instalação:

php --version

Ambientes Locais

  • XAMPP: Windows, Linux, macOS
  • LAMP: Linux (Apache, MySQL, PHP)
  • WAMP: Windows (Apache, MySQL, PHP)
  • Laragon: Windows moderno

Editores Recomendados

  • Visual Studio Code
  • PhpStorm (JetBrains)
  • Sublime Text
  • Atom

Primeiro Programa em PHP

Vamos criar o programa "Hello World" em PHP.

Estrutura básica:

<?php
    echo "Olá, Mundo!";
?>

Arquivo HTML com PHP:

<!DOCTYPE html>
<html>
<head>
    <title>Meu Primeiro PHP</title>
</head>
<body>
    <?php
        echo "Bem-vindo ao PHP!";
    ?>
</body>
</html>

Sintaxe Básica

  • <?php ?>: Tags de abertura e fechamento
  • echo: Imprime texto
  • print: Alternativa a echo
  • //: Comentário de linha única
  • /* */: Comentário multi-linha

Variáveis

  • Começam com $
  • Tipagem dinâmica
  • Case-sensitive
  • Escopo: global, local, static

Variáveis e Tipos de Dados

PHP suporta vários tipos de dados.

<?php
$nome = "João";           // String
$idade = 25;              // Integer
$altura = 1.75;           // Float
$ativo = true;            // Boolean
$dados = array(1, 2, 3);  // Array
?>

Funções em PHP

Funções permitem reutilizar código.

<?php
function saudacao($nome) {
    return "Olá, " . $nome . "!";
}

echo saudacao("Maria");
?>

Trabalhando com Formulários

PHP é perfeito para processar dados de formulários HTML.

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $nome = $_POST["nome"];
    echo "Bem-vindo, " . htmlspecialchars($nome);
}
?>

<form method="post">
    <input type="text" name="nome">
    <input type="submit">
</form>
