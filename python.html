O que é Python?

Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi lançada por Guido van Rossum em 1991.

Características principais:

  • Sintaxe Clara: Foco na legibilidade do código (usa indentação)
  • Multiparadigma: Suporta POO, funcional e procedural
  • Baterias Inclusas: Possui uma vasta biblioteca padrão
  • Interpretada: O código é executado linha por linha
  • Linguagem Geral: Usada em web, dados, automação, etc.

Python é hoje uma das linguagens mais amadas e utilizadas no mundo, especialmente em Inteligência Artificial e Ciência de Dados.

Configuração do Ambiente

Configurar o Python é simples e rápido na maioria dos sistemas operacionais.

Passos básicos:

  1. Baixar o instalador no site oficial (python.org)
  2. Marcar a opção "Add Python to PATH" durante a instalação
  3. Instalar um editor ou IDE (VS Code, PyCharm, Jupyter)
  4. Verificar a instalação via terminal

Verificando a instalação:

python --version
pip --version

Gerenciamento de Pacotes

  • PIP: O instalador de pacotes padrão
  • PyPI: Repositório oficial de bibliotecas
  • Ambientes Virtuais: (venv, conda) para isolar projetos

IDEs e Editores

  • PyCharm (Profissional)
  • VS Code (Leve e versátil)
  • Jupyter Notebook (Foco em dados)
  • Spyder

Primeiro Programa em Python

Python é conhecido por sua simplicidade. Veja o "Hello World":

print("Olá, Mundo!")

Exemplo de Função:

def saudacao(nome):
    return f"Olá, {nome}! Bem-vindo ao Python."

print(saudacao("Desenvolvedor"))

Por que Python?

  • Curva de aprendizado suave
  • Menos linhas de código que Java/C++
  • Comunidade gigantesca e ativa
  • Alta demanda no mercado de trabalho

Onde é usado?

  • Desenvolvimento Web (Django, Flask)
  • Ciência de Dados (Pandas, NumPy)
  • Inteligência Artificial (TensorFlow, PyTorch)
  • Automação e Scripting

Sintaxe e Estruturas

Python utiliza a indentação para definir blocos de código, o que torna o código visualmente limpo.

Variáveis e Tipos

nome = "Python"  # String
idade = 30       # Integer
altura = 1.75    # Float
vivo = True      # Boolean

Estruturas de Dados

lista = [1, 2, 3]    # Mutável
tupla = (4, 5, 6)    # Imutável
dicionario = {"a":1} # Chave-valor
