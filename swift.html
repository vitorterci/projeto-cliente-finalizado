O que é Swift?

Swift é uma linguagem de programação compilada, criada pela Apple em 2014. É projetada para ser segura, rápida e expressiva, substituindo Objective-C no desenvolvimento para iOS, macOS e outros sistemas Apple.

Características principais:

  • Segura: Tipagem forte e eliminação de erros comuns
  • Rápida: Performance comparável a C++
  • Expressiva: Sintaxe clara e moderna
  • Interoperável: Funciona com código Objective-C
  • Open Source: Disponível em GitHub

Swift é a linguagem oficial para desenvolvimento de aplicativos no ecossistema Apple.

Configuração do Ambiente

Para programar em Swift, você precisa de Xcode (IDE oficial da Apple).

Passos básicos:

  1. Instalar Xcode da App Store
  2. Abrir Xcode e criar um novo projeto
  3. Escolher o tipo de aplicativo
  4. Começar a programar

Verificando a instalação:

swift --version
swiftc --version

Plataformas Suportadas

  • iOS: Aplicativos para iPhone
  • macOS: Aplicativos para Mac
  • watchOS: Aplicativos para Apple Watch
  • tvOS: Aplicativos para Apple TV

Ferramentas de Desenvolvimento

  • Xcode (IDE completo)
  • Swift Playgrounds
  • Swift Package Manager
  • TestFlight (distribuição)

Primeiro Programa em Swift

Vamos criar o programa "Hello World" em Swift.

Estrutura básica:

import Foundation

print("Olá, Mundo!")

Para executar:

swift hello.swift

Sintaxe Básica

  • import: Importa módulos
  • print(): Imprime texto
  • //: Comentário de linha única
  • /* */: Comentário multi-linha
  • Sem ponto-e-vírgula necessário

Convenções de Código

  • Nomes de classes: PascalCase (MinhaClasse)
  • Nomes de funções: camelCase (minhaFuncao)
  • Nomes de variáveis: camelCase (minhaVariavel)
  • Constantes: camelCase (minhaConstante)

Variáveis e Constantes

Swift diferencia entre variáveis (var) e constantes (let).

let nome = "João"      // Constante
var idade = 25         // Variável
var altura: Double = 1.75  // Com tipo explícito

// Tipos de dados
let texto: String = "Olá"
let numero: Int = 42
let decimal: Double = 3.14
let ativo: Bool = true

Funções em Swift

Funções em Swift são poderosas e expressivas.

func saudacao(nome: String) -> String {
    return "Olá, \(nome)!"
}

func soma(a: Int, b: Int) -> Int {
    return a + b
}

// Função sem retorno
func imprimir(mensagem: String) {
    print(mensagem)
}

// Múltiplos retornos
func coordenadas() -> (x: Int, y: Int) {
    return (10, 20)
}

Closures

Closures são blocos de código que podem ser passados como argumentos.

let numeros = [1, 2, 3, 4, 5]

// Closure com sintaxe completa
let pares = numeros.filter { (numero) -> Bool in
    return numero % 2 == 0
}

// Sintaxe abreviada
let dobrados = numeros.map { $0 * 2 }

// Closure como argumento
func executar(operacao: (Int, Int) -> Int) {
    print(operacao(5, 3))
}

Desenvolvimento iOS com SwiftUI

SwiftUI é o framework moderno para construir interfaces iOS.

import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack {
            Text("Olá, SwiftUI!")
                .font(.largeTitle)
                .foregroundColor(.blue)
            
            Button(action: {
                print("Botão pressionado!")
            }) {
                Text("Pressione-me")
                    .padding()
                    .background(Color.blue)
                    .foregroundColor(.white)
                    .cornerRadius(8)
            }
        }
    }
}
